## Delay [](#delay)

 
Puts the program to sleep for a specified number of milliseconds. If this
is larger than 1 second, SplashKit will check to see if the user tries to
quit during the delay. If the user does quit, the delay function returns
without waiting.

**Return Type**

- `Void` 

**Parameters**

| Name   | Type                                               | Description                                                                        |
| ------ | -------------------------------------------------- | ---------------------------------------------------------------------------------- |
| milliseconds | unsigned int | The number of milliseconds to wait |

**Signatures**

<Tabs>
  <TabItem label="C++">
```cpp
void delay(unsigned int milliseconds)
```
  </TabItem>
  <TabItem label="C#">
```csharp
public static void Utilities.Delay(uint milliseconds);
public static void SplashKit.Delay(uint milliseconds);
```
  </TabItem>
  <TabItem label="Python">
```python

```
  </TabItem>
  <TabItem label="Pascal">
```pascal
procedure Delay(milliseconds: Cardinal)
```
  </TabItem>

- [**Rnd** (ubound: int, )](/components/sprites/#rnd--3)

:::
