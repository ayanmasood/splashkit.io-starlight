title: Sprites
description: Animations in SplashKit can be used to move between cells in bitmaps and sprites. Each animation generates a number sequence that can then be used when drawing bitmaps.
---
:::tip[Sprites]
SplashKit Sprites allows you to create images you can easily move and animate.

cell information within the bitmap if it is animated at a later stage.

## Sprite [](#sprite)
Sprites combine an image, with position and animation details. You can create a sprite using [`Create Sprite`](/components/sprites/#create-sprite), draw it with [`Draw Sprite`](/components/sprites/#draw-sprite), move it using the [`Sprite Velocity`](/components/sprites/#sprite-velocity) with [`Update Sprite`](/components/sprites/#update-sprite), and animate it using an [`Animation Script`](/components/types/#animation-script).

## Sprite Event Handler [](#sprite-event-handler)
The `spriteeventhandler` function pointer is used when you want to register to receive events from a Sprite.

---

## Sprite Event Kind [](#sprite-event-kind)

This enumeration contains a list of all of the different kinds of events that a Sprite can raise. When the event is raised, the associated `sprite_event_kind` value is passed to the event handler to indicate the kind of event that has occurred.

**Constants**

| Name                  | Description                                     |
| --------------------- | ----------------------------------------------- |
| Sprite Arrived Event  | The sprite has arrived at the end of a move.   |
| Sprite Animation Ended Event | The Sprite's animation has ended.        |
| Sprite Touched Event  | The Sprite was touched.                         |
| Sprite Clicked Event  | The Sprite was clicked.                         |
