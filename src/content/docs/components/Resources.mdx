## Resource Kind [](#resource-kind)
SplashKit is able to manage a number of different kinds of resources for you.

**Constants**

- Animation Resource
  - Animation scripts are loaded as Animation resources. These are located in the project's `/Resources/animations` folder.

- Bundle Resource
  - Resource bundles contain lists of other resources. These are located in the project's `/Resources/bundles` folder.

- Database Resource
  - Database resources are located in the `/Resources/databases` folder.

- Font Resource
  - Fonts resources are located in the `/Resources/fonts` folder.

- Image Resource
  - Image resources are located in the `/Resources/images` folder.

- JSON Resource
  - JSON resources are located in the project's `/Resources/json` folder.

- Music Resource
  - Music resources can be played as music and live in the program's `/Resources/sounds` folder.

- Server Resource
  - Server resources can be sent as responses to web server requests.

- Sound Resource
  - Sound resources can be played as sound effects and live in the program's `/Resources/sounds` folder.

- Timer Resource
  - Timer resources are not saved to file but can be created by SplashKit resource bundles.

- Other Resource
  - Other resources can be loaded directly and will be located in the project's `/Resources` folder.
